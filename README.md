# SABERLite
A Light version of the Simplified and Automated Building Energy Retrofit Tool.

# Tool Description
This is a pre-release version of *SABERLite*. It is intended for student testing and feedback. Students are encourage to run this program on their computers and provide feedback.

Please note that this pre-release version may contain several bugs that are actively being worked upon.

# System Requirements
* Windows (only)
* No python installation required

# Download
To download the tool, please navigate to Releases and download the latest release. You are specifically trying to download the zip file called "SABERLitevx.y.z". Extract the file in your chosen directory and double click on RunSaberLite.exe to run the program. 

Note: Running the tool may take several minutes, please be patient.

# Tool Instructions
To find the detailed instructions on how to run the tool, please navigate to the *docs* folder above.
